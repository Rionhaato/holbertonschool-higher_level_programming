# JavaScript DOM Manipulation

This project covers basic DOM manipulation with JavaScript, event handling, class updates, DOM updates, and data fetching with the Fetch API.

## Files

- `0-script.js`: changes the text color of the `header` element to red using `document.querySelector`.
- `1-script.js`: changes the text color of the `header` element to red when the user clicks on `#red_header`.
- `2-script.js`: adds the `red` class to the `header` element when the user clicks on `#red_header`.
- `3-script.js`: toggles the `header` class between `red` and `green` when the user clicks on `#toggle_header`.
- `4-script.js`: adds a new `<li>Item</li>` element to the list with class `my_list` when the user clicks on `#add_item`.
- `5-script.js`: updates the `header` text to `New Header!!!` when the user clicks on `#update_header`.
- `6-script.js`: fetches the Star Wars character name from the API and displays it inside `#character`.
- `7-script.js`: fetches Star Wars movies from the API and lists their titles inside `#list_movies`.
- `8-script.js`: fetches the French translation of `hello` and displays it inside `#hello`.
