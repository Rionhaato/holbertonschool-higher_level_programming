# JavaScript - Warm up

This project introduces basic JavaScript scripting with Node.js.

## Requirements

- All scripts start with `#!/usr/bin/node`
- Code follows semistandard style with semicolons
- Files end with a new line
- Scripts are written for Node.js 14.x
