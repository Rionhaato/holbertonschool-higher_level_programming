# Python - Classes and Objects

This directory contains solutions for the Python - Classes and Objects project.

Files:
- `0-square.py`: Empty `Square` class definition.
- `1-square.py`: `Square` with private size attribute set at initialization.
- `2-square.py`: `Square` with size validation on initialization.
- `3-square.py`: `Square` with size validation and an `area()` method.
- `4-square.py`: `Square` with a size property (getter/setter) and validation.
- `5-square.py`: `Square` with `my_print()` to print the square using `#`.
- `6-square.py`: `Square` with size and position properties and offset printing.
