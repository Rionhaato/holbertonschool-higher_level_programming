# Python - Input/Output

This project contains tasks for practicing file input/output in Python.

## Tasks

### 0. Read file

Write a function that reads a text file (UTF-8) and prints it to stdout.

File: `0-read_file.py`

### 1. Write to a file

Write a function that writes a string to a text file and returns the number of
characters written.

File: `1-write_file.py`

### 2. Append to a file

Write a function that appends a string at the end of a text file and returns
the number of characters added.

File: `2-append_write.py`

### 3. To JSON string

Write a function that returns the JSON representation of an object.

File: `3-to_json_string.py`

### 4. From JSON string to object

Write a function that returns a Python data structure represented by a JSON
string.

File: `4-from_json_string.py`

### 5. Save object to a file

Write a function that writes an object to a text file using a JSON
representation.

File: `5-save_to_json_file.py`

### 6. Create object from a JSON file

Write a function that creates an object from a JSON file.

File: `6-load_from_json_file.py`

### 7. Load, add, save

Write a script that adds all command-line arguments to a list, then saves them
to `add_item.json`.

File: `7-add_item.py`

### 8. Class to JSON

Write a function that returns the dictionary description of an object for JSON
serialization.

File: `8-class_to_json.py`

### 9. Student to JSON

Write a class `Student` with a `to_json()` method returning its dictionary
representation.

File: `9-student.py`

### 10. Student to JSON with filter

Write a class `Student` where `to_json(attrs=None)` optionally filters returned
attributes.

File: `10-student.py`

### 11. Student to disk and reload

Extend `Student` with `reload_from_json(json)` to replace instance attributes
from a dictionary.

File: `11-student.py`

### 12. Pascal's Triangle

Write a function that returns a list of lists of integers representing
Pascal's triangle.

File: `12-pascal_triangle.py`
