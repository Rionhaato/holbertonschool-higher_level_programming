# RESTful API

This project contains introductory tasks on HTTP, API consumption, and API development using Python.

## Learning Objectives
- Understand HTTP vs HTTPS fundamentals.
- Use `curl` to consume API endpoints.
- Consume and process API data in Python with `requests`.
- Build simple APIs using `http.server` and Flask.
- Implement basic API authentication and security with Basic Auth and JWT.

## Requirements
- Python 3.x
- `requests`
- `Flask`
- `Flask-HTTPAuth`
- `Flask-JWT-Extended`

## Files
- `task_00_requests.md`: Basics of HTTP/HTTPS.
- `task_01_requests.md`: API requests using `curl`.
- `task_02_requests.py`: Fetch and process posts from JSONPlaceholder.
- `task_03_http_server.py`: Simple API using `http.server`.
- `task_04_flask.py`: Simple API using Flask.
- `task_05_basic_security.py`: API security with Basic Auth and JWT.
