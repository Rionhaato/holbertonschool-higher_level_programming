# Python Serialization

This directory contains Python exercises focused on serializing and deserializing data across common formats.

## Files

- `task_00_basic_serialization.py`: Basic JSON serialization/deserialization helpers.
- `task_01_pickle.py`: Object persistence using Python `pickle`.
- `task_02_csv.py`: Serialize and read data using CSV format.
- `task_03_xml.py`: Serialize and read data using XML format.

## Goal

Practice saving and restoring Python data using multiple serialization formats while keeping APIs simple and reusable.
