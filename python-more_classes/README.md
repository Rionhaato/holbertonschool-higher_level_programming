# Python - More Classes

Basics of defining classes and objects in Python.

Files:
- `0-rectangle.py`: Empty `Rectangle` class definition.
- `1-rectangle.py`: `Rectangle` with width and height properties and validation.
- `2-rectangle.py`: Adds `area()` and `perimeter()` methods.
- `3-rectangle.py`: Adds `__str__` to print with `#` characters.
- `4-rectangle.py`: Adds `__repr__` for eval-based recreation.
- `5-rectangle.py`: Adds deletion message in `__del__`.
- `6-rectangle.py`: Tracks instance count via `number_of_instances`.
- `7-rectangle.py`: Supports configurable `print_symbol`.
- `8-rectangle.py`: Adds `bigger_or_equal()` static method.
- `9-rectangle.py`: Adds `square()` class method factory.
