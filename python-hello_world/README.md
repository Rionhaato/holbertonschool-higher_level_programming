# python-hello_world
Introductory Python scripts covering printing, strings, formatting, and style requirements.

## Tasks
- 0. Hello, print: `2-print.py` prints a quoted sentence exactly as required.
- 1. Print integer: `3-print_number.py` prints an integer using f-strings without casting.
- 2. Print float: `4-print_float.py` prints a float with two decimal places using f-strings.
- 3. Print string: `5-print_string.py` prints a string three times, then its first 9 chars.
- 4. Play with strings: `6-concat.py` builds and prints "Welcome to Holberton School!".
- 5. Copy - Cut - Paste: `7-edges.py` slices a word into first 3, last 2, and middle.
- 6. Create a new sentence: `8-concat_edges.py` uses slices to output a full sentence.
- 7. Easter Egg: `9-easter_egg.py` prints "The Zen of Python" via import.
