# Python - if/else, loops, functions

Introductory exercises for Python control flow (if/else), loops, and functions.

## Task 0: Positive anything is better than negative nothing

Write a script that assigns a random signed number to the variable `number` and
prints whether it is positive, zero, or negative.

File: `0-positive_or_negative.py`

## Task 1: The last digit

Write a script that assigns a random signed number to the variable `number` and
prints the last digit of the number with a message based on its value.

File: `1-last_digit.py`

## Task 2: I sometimes suffer from insomnia. And when I can't fall asleep, I play what I call the alphabet game

Write a program that prints the ASCII alphabet in lowercase, without a new
line, using only one loop and one `print` with string format.

File: `2-print_alphabet.py`
